Hay que subir los datasets a Vertex para poder hacer modelos de AutoML o modelos Customs
Antes de submitear un custom training job, hyperparameter tunning job, or training pipeline to Vertex AI. 
Es necesario crear una aplicación de entrenamiento en python (python training aplication) o 
un custom container para definir el codigo de entrenamiento y las dependencias que quieres crear en Vertex AI. Hay dos opciones para crear un container:
1. Usar un pre-builit container de google. (realizar un entrenamiento y predicción rápido)
2. Usar tu propio container image.*:
2.1. Python training script 
2.2. Dockerfile: Codigo del modelamiento, dependencias e instrucciones de ejecución.
Se crea el contenedor con cloud build y se hace el push del container en GCR (Google Container Registry)
Se realiza el modelamiento de esta forma con la finalidad de empaquetar el codigo de entrenamiento y dependencias para usarlo como pare te de un Pipeline de Vertex.
